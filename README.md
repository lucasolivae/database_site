# Database Site - Flask

Um projeto web simples desenvolvido em Python usando Flask.  
Este sistema permite cadastrar clientes em um banco de dados SQLite e visualizar suas informações.

## O que este projeto faz
- Cadastra clientes (nome, data de nascimento e telefone)
- Gera automaticamente um ID alfanumérico para cada cliente
- Lista todos os clientes cadastrados
- Permite acessar uma página individual para cada cliente
- Armazena os dados em um banco SQLite

## Como usar

### Requisitos
- Python 3.x instalado
- Flask instalado

### Instalando dependências
pip install flask

### Executando o projeto
1. Abra o terminal na pasta do projeto  
2. Execute o comando:
python codigo.py

3. Acesse no navegador:
http://127.0.0.1:5000
