# Database Site - Flask

A simple web project developed in Python using Flask.  
This system allows you to register clients in a SQLite database and view their information.

## What this project does
- Registers clients (name, birth date and phone)
- Automatically generates an alphanumeric ID for each client
- Lists all registered clients
- Allows access to an individual client page
- Stores data in a SQLite database

## How to use

### Requirements
- Python 3.x installed
- Flask installed

### Installing dependencies
pip install flask

### Running the project
1. Open a terminal inside the project folder  
2. Run:
python codigo.py

3. Open in your browser:
http://127.0.0.1:5000
